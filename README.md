# Books
A simple books CRUD app on golang using repository pattern
## Tools needed to be preinstalled
- Docker
-  go
- make
## How to run?
1. copy .env.example to .env with your envs
2. Run with "make"
   - To run in debug mode do
     ```shell
     make dev
     ```
   - To run in release mode do
     ```shell
     make run
     ```
